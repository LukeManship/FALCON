Quick Start Guide
=================

Installation
____________

.. code-block:: bash
git clone git://github.com/PacificBiosciences/FALCON-integrate.git
cd FALCON-integrate
git checkout master
make init
source env.sh
make config-edit-user
make -j all
make test  # to run a simple one
